听课笔记 - 袁越：科技如何拯救世界

发表于 2018-05-13 | 分类于 听课笔记 | 阅读次数

说明：这是一种新的内容类型，主要记录我听到的一些课程或讲座内容，结构稳定，主题多样，希望能以此记录我对这个世界认知的各种大小变化。

基本信息

主题：科技如何拯救世界 —— 专访盖茨的启示
嘉宾：三联周刊主笔 袁越
地点：前途驿智趣沙龙

主要内容

袁越老师采访了盖茨基金会创始人比尔盖茨，并围绕着盖茨关注的健康、新能源、转基因等话题与大家作了一个多小时的分享。

1. 了不起的盖茨

这次专访最后在三联生活周刊成稿，并作为封面主题：富人的药方。这个世界最大的基金每年会将 40 亿美金的利息用于慈善事业，并以倒计时的方式运转，旨在解决世界级的一些让人头疼的问题。

这个基金会的创始人：微软创始人、前世界首富比尔盖茨是个大家既熟悉又陌生的存在，但近些年他以「读书狂魔」的身份被更多国人重新认识，每周在博客分享的书籍成为很多人的阅读参考。

2. 伟大的疫苗

盖茨基金会投资了很多项目，其中的很多是高新技术企业，虽然不再担任微软的 CEO，但毫无疑问，盖茨是一个绝对的技术派，他相信技术的力量，也推动发明了 Arktek 疫苗存储保温桶等产品，在贫困地区发挥着重要的作用。

这个世界有两样东西拯救了最多的人命，一样是疫苗，一样是抗生素。

盖茨是一个疫苗的拥趸，他希望并且在努力通过疫苗解决已知的一些现实问题：比如小儿麻痹。虽然小儿麻痹现在只在巴基斯坦等少数国家存在，但整个世界依然还需耗费大量成本为所有新生幼儿接种。没有哪个国家的政府愿意花钱去彻底消灭这种病毒，而这也许只能由比尔盖茨这样的人推动。为什么盖茨愿意花费那么多钱去消灭一种其实杀伤力并不大的病毒呢？

因为对整个人类更合算。

从经济学的角度看，现在投入成本预防或者解决问题和等问题发生之后处理问题，两者的成本比较非常关键，它直接决定了是否在现阶段的政策。虽然小儿麻痹对人类的杀伤力并不大，但如果能彻底消灭，那么全人类都将省去继续接种疫苗的成本，这会涉及到几亿，几十亿的人口，是一件非常了不起的事情。

关于疫苗，国内有些言论说，一些疫苗都会让中国人先做实验，但事实上，目前几种主流的疫苗都是在美国和前苏联征集了几百万甚至上千万儿童实验得到最后确认的，我们也许不仅不应该质疑，反而应该对此表达感谢。

话说回来，如果说这个世界上哪个问题最大，除了全球变暖之外，还有就是人口问题，这么看来计划生育似乎是个好政策，而现在的二胎开放也许还需要认真审视后续的进展。

2. 转基因与环境保护

转基因技术本身能够带来很多立竿见影的效应，不仅仅关乎经济利益，还对环境保护有诸多帮助。有个非常典型的例子，很多渔民捕捞并不会把刚刚生出的小鱼仔放回，而是把它们做成鱼饲料，因为养殖鱼必需的一种元素，奥米茄 3 只有在这些海鱼里才存在。事实上，转基因技术已经成功做出含有这种物质的玉米，这直接可以导致渔民不再需要竭泽而渔，让更多小鱼仔长大，但直至如今尚未能引入，诸多遗憾。

包括张启发的农业技术也是建立在大量转基因技术之上，这也许才是中国农业的未来。

3. 气候变化是人类面临的最大威胁

我们对气候变化存在误解，很多人会因为近期气温的下降或者局部地区的极寒而怀疑气候变暖并不一定是真实威胁。其实，气候变化最大的问题并不是来自于升温，而是来自于升温的速度，在之前，可能需要几万年才会升高一度，现在可能 100 年、50 年就会升高 1 度，这导致很多物种无法快速适应，进而灭绝。

气候问题是一个绝对的工地悲剧，也可能让人类像温水煮青蛙一样后知后觉中错失解决问题的机会。

盖茨认为，解决这个问题最好的方式是：核能。其实地球上所有的能源可以说都是太阳给予的，如果能够突破核聚变技术，可能真正意义上解决能源问题。

3. 如何让世界变得更好

读万卷书：掌握权威主流的观点和看法

行万里书：强迫你面对你不愿意面对的现实

实用主义：至高境界，不用谈什么理想，先做出来！

不怕失败：不怕否定过去

乐观主义：相信所有问题终将解决

达尔文是对的：人类就是靠不断试错、修正向前发展。

4. 其他

4.1 尊老爱幼的必然进展在古代，一个人的见识和年龄直接成正比，年轻人绝无可能在知识和见识层面超越长者，因此长者必然得到尊重，而互联网的出现，打破了这样的格局，在这个时代，更多年轻人的见识远大于老年人，因此尊老就不太被重视。同样，在古代，很多孩子因为种种疾病活不到成年，非常脆弱，所有人都需要爱护这些孩子，但如今，随着医疗技术的发展，儿童的存活率已经大幅度提高，大家对熊孩子的忍受度也便大幅降低了。

4.2 精准的时间感比尔盖茨的访谈日程安排在 12:25，精确到分钟是这个精力充沛、「张牙舞爪」的聪明人的时间精度。

最后的话

这个世界很大，也很酷。

了解这个世界的问题，去获取和评判当下解决问题的思路和做法，即便不能投入其中，也可以表达关注。

这个世界有很多病，比尔盖茨开了他的药方，未见得一定全部正确，但它值得我们去思考和研判，也许这个世界真的能被治愈，像消灭天花病毒一样，人类的灾难都将在「达尔文的预测」中作为试炼去磨难人类，也终将被人类攻克。

这个世界应该会更好，我觉得。
