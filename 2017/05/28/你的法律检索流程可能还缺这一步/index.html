你的法律检索流程里可能还缺这一步

发表于 2017-05-28 | 分类于 律师效能手册 | 阅读次数

律器的第三期，我们来给大家介绍印象笔记强大的搜索功能，同时也跟大家聊一聊：新时代下，我们该怎样做好搜索这件小事。

有人就问了搜索，不就是输入框里输个关键词么，有必要太当回事么？
答案是：有必要，而且越来越有必要！这是互联网信息爆炸以及律师工作自身特点所共同决定的。

屏读时代的必然要求

对一位科学家来说，整理和组织材料的过程本身，就是一种享受。……从事这一工作时，感到很愉快。这种快感，实际上就是意义本身。——《奇特的一生》

是的，任何一个有过将浩如烟海的法律法规、卷宗材料、财务报表、访谈笔录梳理出脉络经历的律师都一定品位过相同的感受。我们工作的主要内容就像人类之于知识的分类一样，将庞杂的信息按照新的维度重新编排，形成组块、框架、链条，在精密的逻辑指导下，匹配、反驳、印证，很辛苦，但也很有趣。

知识分类的开始在人类历史上是具有划时代意义的，当人们开始根据特定的需要和标准，通过比较，把人类的拥有的全部知识按照相同、相异、相关等属性划分成为不同类别的知识体系开始，人类便拥有了更高级别驾驭管理知识的能力，并有机会不断突破认知的边界。

随着知识总量的不断扩大，人类对知识的分类技术也越发成熟

单一学科内的知识分类也在不断细化

我无比崇拜「分类」，觉得这件事情充满了人类无以伦比的智慧，能够做好这件事的前提一定是对某个领域的知识达到一种全息的理解和认知。但随着知识增量的不断增加，知识的裂变和衰减都在持续加速，知识多维度、跨学科的趋势更加明显，这都导致知识的分类变得越来越困难。现阶段几乎无法避免的，人类对知识的掌控将发生新的变化：

混乱→有序→在框架下有秩序的混乱。

当知识的总量多到一定程度，分类的作用在边际效应上就会逐渐递减。原因很简单：

  • 分类少，每一个类别涵盖的知识总量还存在大量合并的空间，说白了，就是一级目录太少，二、三级目录太多；
  • 分类精密一些，往往分类本身又成为我们知识管理最首要的障碍。类似电脑文件夹的层级，一旦超过三层，其打开率将骤减30%，同样的道理，我们把有价值的内容藏得越深，我们再次打开的概率也就越小。

作为律师，我们当然要追求秩序，但对待所有的信息和知识，如果都希望使其最末端也建立起完美的秩序，那么，毫无疑问，这个过程将耗费你巨大的心力，这虽然不能说毫无意义，但性价比一定是极低的。换言之，分类的目的是为了能够建立秩序并在秩序中提升知识被提取复用的概率，进而解决问题或者创造更多有价值的知识。分类和建立秩序只是其中环节，而不是我们知识管理的终点。完美的分类是需要多个层级不断打磨调整的，这个过程其实很有趣甚至让上瘾。这时候就要多遵循二八法则，要克制！适当的时候，要强迫自己接受一种「在框架下有秩序的混乱」，但这需要一个过程，一开始并不容易。（这也是Evernote和OneNote背后思维方式上最本质的差异性）

律师工作的核心能力

一项看起来简简单单的法律调研，对于律师而言，可能需要去检索法律法规、相似案例、专业指引，要从大脑、电脑、网页、微信、邮箱、笔记本、书籍等信息载体去搜罗海量数据，并从中寻找解决问题的答案。没有一个强大的搜索能力，律师的一天可能始终在艰苦卓绝的搜索中捱过。

毫无疑问，搜索是律师工作的必备技能，很多业务技能的提升和工作任务的完成，都以搜索为基础的。搜索能力是一个律师业务综合能力的体现，更是律师专业化的「加速度」。在前几日结束的无讼有声中，蒋勇老师提到「要给律师赋能」，能力的掌握需要时间的积累，有些急不得，也不用急，而有些「能」则需要抓紧时间、尽早掌握。

喜欢金庸的朋友一定听说过小无相功，掌握小无相功，就可以运转少林七十二门绝技、天山折梅手、大力金刚指，而搜索能力某种意义上便是律师修炼的小无相功。

印象笔记的搜索是什么搜索？

现在有越来越多的工具可以帮助我们提高法律检索的效率，比如无讼案例、威科先行、北大法宝、律商网等等。这些工具的逻辑很好理解，那么，说了这么多搜索，跟印象笔记这么一个云笔记软件又有什么关系呢？

答案就是：仅仅能搜到是不够的，你需要搭建自己的私人Google，享受极致的搜索效率

我们随随便便的一个搜索返回的结果，可能都是毫秒级别超过7位数的条目，最终呈现一种：我不知道在哪一页，但我知道这里面一定有我要的结果的局面。但事实上，如果总数原先以亿计，搜索之后得到一个万计的结果，对我们而言其实意义是不大的。

在我看来，随着我们学习工作的经验积累，最高效的搜索就是不搜索，而更多从我们自己的知识储备和过往经验中寻找答案。好比一个十几年的律所，如果实习生起草一份最基础的合同，还需要去百度文库下载模板的话，那么我劝这个实习生还是尽早考虑换所比较明智。

知识和经验快速积累的一个重要前提就是知识和经验的有形化，当你能够让所有的努力有形，才可能对其进行量化，并持续地、有条不紊得展开追踪。而这一切，印象笔记都可以帮助你实现。当我们所有阅读过的干货文章、做过的大小项目、听过的批评教诲、解决过的法律难题、搜罗来的法律模板都存储在自己印象笔记中，你的搜索一定不会从漫无边际的互联网开始，而是从自己的知识数据库出发，在当下所有法律检索的教程中，我认为都应该增加一条：检索自己及团队的Evernote知识库，因为这里面有你和你的团队几乎所有的努力和积累，这是互联网上绝对找不到的宝藏。

因此，只要手机在手，我们可以随时调取法条、案例、指引、模板，几乎分秒间呈现给客户、老板、同事……相信我，这种体验非常美妙。

印象笔记：我能给你谷歌级别的搜索体验

1.全局范围的标题搜索

使用苹果电脑的小伙伴或多或少使用过Spotlight，不管你想打开程序或者文件，只要快捷键呼出窗口，几个字母敲进去就能实现秒开。

这种操作感受非常好，比起逐层打开文件夹去双击打开的方式，速度要快上几倍不止。

类似的，印象笔记也有这样的功能，不论你在软件的什么位置，只要你按下Com+J，呼出标题搜索窗口，输入关键词，点击回车，2秒钟就能让你快速进入目标笔记本中，整个操作行云流水，真正随时随刻调取想要的内容。

在使用一段时间Mac和Evernote之后，我渐渐发现，自己90%以上的文件和程序的开启都是通过搜索而非快捷方式，这是一种非常高效的行为模式的转变，这也为我节省了大量是时间。

2.无所不能的全文搜索

印象笔记有非常多的使用场景，前几日听李笑来在讲座中分享了他的用法，非常的简单暴力：

我觉得不错的东西，就扔到印象笔记里面，我也不整理，但是我有一个基本的印象，我需要的时候可以搜索到。

虽然，我的用法没有李笑来老师这么粗犷，我还只能接受「在框架下的无秩序的混乱」，但是这也从侧面说明印象笔记强大的全文搜索能力。能够实现全文检索对于律师工作来说是意义非凡的，且不要说Win系统中仅仅是标题搜索，那个体验能有多差，在印象笔记中，不仅可以全文搜索，而且速度快到你根本感受不到有搜索的过程。

比如，我前段时间听了何帆大法官的知乎live，有一次跟律所小伙伴聊到研读美国经典判决书的问题，想要给他分享何帆老师在讲座中的一个观点，于是打开印象笔记，只输入「何帆」、「听课」两个词

就可以从我几千条笔记中瞬间调取当时的听课笔记，这比再打开知乎live的链接，从近2个小时，上白条录音中，找到多少分钟的某一句话要高效不知道多少倍。

3.高级搜索语法

印象笔记的搜索功能强大到让你有Google一样的搜索体验。它看起来可能只是一个简单的文字搜索区域，但它的高级搜索语法允许你通过笔记的标签、创建日期、所含媒体类型（音频、图片等）、是否包含复选框、是否有加密信息、甚至它们被创建的地点来搜索。

我们来总结一下，在海量的知识面前，试着放弃绝对完美的分类，接受一定程度的无秩序，因为，只要你拥有了Evernote，所有的努力都能够得到沉淀，任何信息，你都可以在毫秒间搜索提取，并为你创造价值。
