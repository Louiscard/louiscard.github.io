律师效能手册 - 你需要一个Inbox

发表于 2017-05-23 | 分类于 律师效能手册 | | 阅读次数

我常跟律师朋友推荐好用的生产力工具。

“印象笔记特别好用！”
“嗯嗯，我也在用”
“你有什么独特的使用场景么？”
“嗯，我偶尔记个日记”
“额……”

每次听到这样的回答，我都忍不住扼腕叹息。只用印象笔记记录心情，虽然不是不行，但着实有些大材小用。不过，接触多了我渐渐发现问题所在，很多律师兴冲冲的下载安装注册，打开传说中的知识管理神器，看着白茫茫一片，什么都没有，也不知道要新建什么，于是，记点心情啊，放点杂事啊，最终不了了之。

所以，律师效能手册，从这周开始陆续更新「印象笔记 - 律师必备哪些笔记本」系列，来教大家在知识管理工具，特别是云端笔记上，到底应该新建哪些律师必备的笔记本。

1 用GTD优化你的信息输入工作流

工作就是信息流，只是有些领域的平缓，有些行业湍急。律师行业基本上属于泄洪时候的大坝和黄果树大瀑布的级别，数据处理带宽需要不停地升级套餐，从10兆到百兆到前兆，没有尽头。

我们别无选择，必须要不断思考：怎样才能轻松驾驭这滚滚的信息流。

1.1 一切皆是学习

很多人标榜自己是「终身学习者」，以彰显自己积极向上、渴望进步的形象。律师行业，鲜有人能抗拒这个身份，对律师而言，似乎没有其他选项，唯有时刻学习这一条艰苦道路，只顾日夜兼行。

同时，如今学习的外延已经扩大很多，早就不是我们传统的坐在书桌钱啃一本书，或者坐在教室里听一堂课啦。任何将信息转变为知识内化为能力的都可以称之为学习。所以，从一篇文章、一节微课、一次讨论、一个灵感到内化成你的一条案件处理经验，这个过程，知识管理能力或者说信息流处理能力就存在巨大的差异啦，有些律师做得简洁高效，有些律师不仅繁琐，期间还丢失很多重要信息。

所以，一个卓越的律师，一定也是一个信息流的处理高手，他像一个神奇的魔术师，信息在他的调配下，顺从地从各种源头流入他的宏大壮阔的知识体系中。

1.2 程序之美

成为一个优秀的魔术师，关键是在「见证奇迹的时刻」之前，把一切流程都安排妥当，怎么把兔子从暗箱腾挪出来，藏在幕布之后，再华丽地跳到观众眼前，一边吸引大家的注意力，一边让兔子又在观众面前消失，再爬出一只乌龟。

最炫目的当然是舞台上的表演，但最有技术含量的是对整个魔术所有环节的精心设计，只有精准的设计才能让所有巧妙机关在舞台上自动运转，博得掌声。同理，律师知识管理中最有价值的部分，同样也是设计流程。如何根据自己的工作场景和具体需求，设计一个从信息输入到输出的完整闭环，让整个流程能够严丝合缝，甚至充满美感，像一个设备制造的流水线，一切都自动运行，一切都恰到好处，没有一丁点额外的浪费。系统能偶不断提取信息，加工成知识，充实到自己的知识体系中，一切显得轻松又和谐。

1.3 借鉴GTD的基本框架

研究过时间管理的小伙伴肯定听说过GTD，有非常多的高效人士在以此打造自己的任务管理系统。作为一个资深的GTD的践行者，可以负责任的表示：「亲测有效」，这几年来的贯彻执行下来，切实提升了自己工作和生活的效率。不过，研究得越深刻越能体会到：感觉高高在上的GTD系统，本质上就是一个叫做Allen的魔术师设计的一整套流程。

只是这个工作流上处理的是项目和任务，而知识管理系统所打造的流程用来传输更加丰富的工作层级的信息。虽然承载的内容不同，但是基本方法是类似的，我们可以试着以备受全世界高效人士推崇的GTD流程为基础，打造我们律师行业的知识管理工作流。

2 第一个建立的笔记本：Inbox笔记本

我们从上面这张图能够看到，在GTD的流程中第一个环节叫做「捕获」，简单说就是把所有信息，先一股脑地放到收件箱中。这个「收件箱」就是我们今天的主角。其地位之高，甚至重要到可以称之为「收件箱思维」。

相对于传统的知识管理，我们大多数时候看到一个有用的信息，就直接在目标位置新建一个Word文档，然后把内容拷贝进去并进行保存。但我们所有人都知道，提高效率有一个最基础的原则就是——批量处理。相对于将所有输入的信息直接存入目标位置而言，先进行低成本的收集，将其放入一个地方（收件箱），然后在固定时间段，批量处理这些信息并将其移动到相应的地方。这两种方式看起来类似，但实践起来，先收集和再批量处理信息的效率比传统方式要高出好几个level不止，更重要的让你的输入成本变低，输入的颗粒度变高，因为所有信息都归入一个笔记本，也会大大减少对信息的焦虑。

我们今天要建立的，就是这个让我们可以愉快输入信息的容器——Inbox，作为信息的入口和中转站，inbox里放着你每天最有价值的信息，从这里分发出去的信息将归入到你的知识系统，供你今后在工作中调取复用。

3 新建印象笔记的收件箱

讲了这么多理念，我们打开印象笔记，一起来新建我们的第一个收件箱吧。

3.1 新建笔记本

方法一：打开印象笔记的笔记本视图，左上角就有一个新建笔记本的按钮，点击即可新建笔记本。

方法二：点击菜单栏，文件 | 新建笔记本 | 选择私人笔记本。

最后，只需要键入名称，选择保持私有，点击新建笔记就可以啦。

3.2 新建笔记本组

知道了如何新建笔记本之后，很多小伙伴兴冲冲地新建了好几个笔记本，然后希望把他们都合并到一个笔记本组里，怎么做呢？方法非常简单，一个字：拖！

后面我们会讲到，有一些平台，比如微信公众号、微博、kindle的书摘，剪藏到印象笔记会自动生成一个笔记本，所以，一个inbox笔记本可能无法满足我们，我们就需要把这些拖放到一起形成一个Inbox笔记本组。

3.3 重命名笔记本

如果你想给笔记本或者笔记本组换一个名字，那就需要重命名笔记本，方法也非常简单，在笔记本或者笔记本组上右键，选择重命名即可。

4 告别OW时代的第一步

我相信，读到这里，你已经迈开了构建知识管理工作流的第一步。接下来，我们将继续一步一步地装载齿轮、调测机器，用不了多久，当你不知不觉成为一名技术派律师，能够驾驭更多生产力工具帮助自己高效工作，以及建构完成自己的知识管理系统的那一天，你就文中提到的在舞台上可以让所有观众为之喝彩的魔术师。

今后，我也会持续给大家分享各种思维、方法和工具，是时候，告别「OW」（Only Word）时代，开始接受一些新的生产力工具帮助自己提升工作效率，成为更了不起的法律人啦。
