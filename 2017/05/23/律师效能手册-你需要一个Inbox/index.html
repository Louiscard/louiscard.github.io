律师效能手册 - 你需要一个Inbox

发表于 2017-05-23 | 分类于 律师效能手册 | | 阅读次数

我经常跟律师朋友推荐好用的生产力工具，经常会有这样的对话：

“印象笔记特别好用！”
“嗯嗯，我也在用”
“你有什么独特的使用场景么？”
“嗯，我偶尔用它记记日记”
“额……”

每次我听到这样的回答，我都忍不住遗憾，只是用印象笔记记录个心情，虽然不是不可以，但是实在是有点浪费，不过反过来我也能够理解，打开传说中的神器，发现里面白茫茫一片，什么都没有，都不不知道要建立什么东西。

所以，律师效能手册，从这周开始陆续更新「印象笔记 - 新建什么笔记本」系列，来教大家在云端笔记，特别是印象笔记上，到底应该新建哪些必须的笔记本。

用GTD流程优化你的信息输入

每天的工作都是信息流，只是有些工作的信息流平缓，有些行业湍急一些。律师行业基本上属于泄洪时候的大坝和黄果树大瀑布，每天的数据处理带宽变得越来越大。

我们别无选择，必须要考虑，怎样样才能让这些驾驭这滚滚的信息流。

信息流

很多人的座右铭是「终身学习者」，用来标榜自己积极向上，渴望进步的价值观。律师行业，少有这样标榜的，因为对律师而言，似乎没有其他选项，你只有时时刻刻去学习这一条道路可走，只能一条道走到黑。

不过，学习的范围也越来越宽泛，早就不是我们传统的坐在书桌钱啃一本书，或者坐在教室里听一堂课才叫学习啦。任何将信息转变为知识内化为能力的都可以称之为学习。所以，从一篇文章、一节微课、一次讨论、一个灵感到内化成你的一条案件处理经验，这个过程，有些律师可以做得简洁高效，有些不仅繁琐，中间甚至还要丢失很多重要信息。

一个卓越的律师，一定也是一个信息流的处理高手，他将像一个魔术师，信息在他的支配下，顺从的从千头万绪流入他宏大壮阔的知识体系中。

程序之美

不过成为一个魔术师也不是一件不可能的事情，关键是你要在「表演」之前，把一切流程都安排妥当，然后在观众面前让所有环节都有条不紊。魔术师要设计怎么把兔子从暗箱里腾挪出来，藏在幕布后面，华丽地跳到观众眼前，再吸引大家的注意力，然后让这只兔子在观众面前消失。

最有技术含量的并不是在舞台上的表演，而是对整个魔术每一个环节的设计，只有精准的设计才能让整个环节在舞台上自动运转，博得掌声。因此，律师的知识管理最有价值的部分，同样也是设计流程，如何结合自己的工作场景和需求，设计一个从信息输入到输出的完整闭环，让整个流程能够严丝合缝，甚至充满美感，像一个机器流水线，一切都自动运行，一切都恰到好处，没有一丁点的额外的浪费，系统不断提取信息，加工成知识，充实自己的知识体系。

向GTD学习

研究过时间管理的小伙伴肯定听说过GTD，世界上有非常多的高效人士都在按照GTD方法打造自己的任务管理系统。作为一个资深的GTD的践行者，可以负责任的表示确实「亲测有效」，切实提升了自己的时间管理能力。

其实，感觉高大上的GTD本质上就是一个精心设计的流程。

只是这个工作流上处理的是任务和项目，而知识管理系统所打造的流程用来传输更加丰富的信息。虽然承载内容的品类不同，但是基本方法是类似的，我们可以全世界高效人士推崇的GTD流程为基础，打造我们的知识管理工作流。

第一个建立的笔记本：Inbox笔记本

我们看到在GTD的流程中第一个环节「捕获」，就是把所有信息，先放到收件箱中。

终于到了我们今天的主角了，那就是「收件箱」，收件箱在整个流程中非常重要，甚至重要到可以称之为「收件箱思维」。相对于传统的知识管理，我们大多数时候看到一个有用的信息，就直接在目标位置新建一个Word文档，然后把内容拷贝进去并进行保存。然而，我们都知道，提供效率有一个最基础的原则就是——批量处理。

相对于将所有输入的信息直接存入目标位置而言，先进行统一的低成本的收集，将其放入一个地方（收件箱），然后在一个时间段，批量处理这些信息并将其移动到相应的地方。这两种方式看起来类似，但是真正实践起来，时时刻刻的收集和批量处理后的分流在对信息处理上的效率比传统方式要高出好几个level。

而我们今天要建立的，就是这个让我们可以愉快输入信息的容器——inbox，作为信息的入口和中转站，inbox里放着你这一天最有价值的信息，从这里分发出去的信息也将归入到你的知识系统中，供你在今后的工作中调取复用。

如何操作

讲了这么多理念，我们来打开印象笔记，一起来新建我们的第一个收件箱吧。

新建笔记本 & 新建笔记

方法一：打开印象笔记的笔记本视图，左上角就有一个新建笔记本的按钮，点击即可新建笔记本。

方法二：点击菜单栏，文件 | 新建笔记本 | 选择私人笔记本。

最后，只需要键入名称，选择保持私有，点击新建笔记就可以啦。

新建笔记本组

那知道了如何新建笔记本之后，很多小伙伴兴冲冲的新建了委托合同、承揽合同、技术合同等一系列笔记本，然后希望把他们都合并到一个笔记本组里，怎么做呢？方法非常简单，一个字：拖！

后面我们会讲到，很多平台的内容到印象笔记会自动生成一个笔记本，所以，到后期，我们可能就不仅是一个inbox，而是一个inbox组。

重命名笔记本

如果你想给笔记本或者笔记本组换一个名字，那就需要重命名笔记本，方法也非常简单，在笔记本或者笔记本组上右键，选择重命名即可。
