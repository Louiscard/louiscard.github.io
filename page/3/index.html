低水平英语学习者的高效学习道路

发表于 2017-07-18 | 分类于 英语 | 阅读次数

由于工作的需要，需要接触不少英文资料，有时候还需要和国外的一些合作伙伴和客户进行简单的邮件沟通，本来以为自己的英语水平差不多够用，然而，事实上，就算是有Google Translate，还是，觉得……不行啊……

于是，这几个月对英语学习又提高了思想认识，也总结出了一些我认为比较好用的经验，今天跟大家做一个简单的分享。

1.通过speaky等平台找个跟你交换语言的英美小伙伴；

2.确定每天外文信息源，我自己主要是订阅了几个博客还有用calibre下载Economist；

3.每天挑一篇文章重点阅读，下划线标注陌生单词、highlight经典的句子、红色标注不懂的句子；

4.选择性的查询单词然后将其录入到旁边的Anki（一款记忆辅助工具）,每天抽空背一背；

5.把整篇文章粘贴到Google Doc（或者「一起写」），分享链接给你的小伙伴，让他帮忙解释下红色的一些句子，以及讨论下这篇文章的关键章节。

以上，重点就是对于张口喜欢说「my english is very poor」的我们，每天坚持阅读并且用科学的算法背诵自己专属的单词表的确提升很高的效率，希望对你有帮助。

成长型律师团队，从打造复盘系统开始

发表于 2017-07-06 | 分类于 律师效能手册 | 阅读次数

前段时间在无讼阅读上看到梁枫律师关于复盘的文章，收获颇丰。这一期的「律师效能手册」就借此来谈一谈：如何用工具打造复盘系统，帮助团队跨入高速成长的快车道。

重新认识复盘

好多词我们都听说过，比如GTD、区块链、罗伯特议事规则……但好多词，我们似乎只停留在了「听说过」。所以，打造复盘系统之前，咱们先一起重新认识下复盘这件事。

复盘大师阿尔法狗

复盘这个词最早来源于围棋，指一盘棋下完，两人重新打谱，检查双方攻守漏洞，以期精进。说到围棋，就不得不提前阵子著名的人机大战，在大败世界排名第一的棋手柯洁之后，阿尔法狗已经成为围棋界名副其实的独孤求败。

那问题来了：

  • 首先：为什么阿尔法狗能这么强大，可以将专业训练几十年的人类棋手斩落马下？
  • 其次：计算机技术发展了70多年，计算能力早就达到了几亿亿次/秒，为何直到今日方才大显神威？

答案就是这几年来深度学习技术（Deep Learning）的日益成熟，其通过建立、模拟人脑学习的神经网络，用非监督式的特征学习替代了低效的手工特征获取，大幅提升了人工智能的发展速度。

简单来说就是：阿尔法狗可以在既定规则内自我训练，不断分析决策优劣，洞察更底层的逻辑关系，从而突破原有认知，打破传统棋路，获得棋术提升，最终碾压人类棋手，甚至被柯洁称为「围棋上帝」。

所以说，大家都是刻意练习，人家一天下几千盘，每一步都分析总结并优化算法，在这样的复盘大师面前，人类下不过AI实属正常。况且有哪个棋手可以做到每一盘、每一子都能精准记忆、反复推演、复盘内化？

所以说，不知疲倦的高水平练习，让努力能够积累的复盘系统，才是阿尔法狗能够短时间成为围棋界Top 1 的致胜法宝。

你所谓的执行力，只是偷懒的借口罢了

发表于 2017-06-19 | 分类于 时间管理 | 阅读次数

初入职场，大概都听过这样的话。

「小吕，没事帮我收个快递」
「小吕，给你邮箱，催下设计那边的进度」
「小吕，给你个电话，去机场接下周总」

因为这样的任务常和便利签同时出现，所以，处在这个阶段的员工，常被亲切地称为 —— 「便利签青年」。

刚进公司，什么都不懂，拼得就是姿态低、执行力强！甭管什么脏活累活，拿到任务，当然是立即执行。一天风风火火、跑前跑后，觉得自己真是一个积极向上、追求进步的好员工啊。

然而，随着经验积累和新人加入，终究有一天会结束这样的小白状态，除了事务性工作，手头也开始负责起一些小项目。有了越来越多自主安排的时间之后，就需要给手头的事情排排序，分个轻重缓急了。

可是，事情越来越多，时间越来越少，感觉自己像是被放出肖申克的老布，面对大把自由支配的时间，竟忍不住怀念起那些拿着便利签跑来跑去的日子。甚至，无法完成身份转变的年轻人，最终沦为自己的「便利签青年」，身陷整日忙碌的牢笼中，无法自拔。

站在社会分工细化的趋势下看问题

李丰老师开场先提出一个概念 ——「社会分工细化」，意思是各行各业，特别是提供职业技能的服务行业会发生越来越深刻的分化和区隔。

我们用美国设计的手机、看印度拍摄的电影、开德国制造的汽车……享用整个世界提供的产品和服务，这本身就是社会分工最生动的体现。

那么，这个听起来让所有人受益的玩法，在法律服务行业是否同样有效呢？

53号纪录片 | 学术研究之外，我还有个「私人图书馆」

发表于 2017-06-12 | 分类于 53号纪录片 | 阅读次数

从今天起，我会在一些时刻，像古希腊的说书人一样，从一个村庄云游到另一个村庄，用Odysseus海上探险的故事吸引村民的关注和打赏。

有趣的灵魂不该只是个ID

自去年入驻在行，有机会结识了很多有趣的人。有在美国执业的心理咨询师、热衷慈善的房地产商、认知心理学博士后、空军副团级军官、华尔街日报记者等等等等。大家找我多是想聊一些知识管理和生产力工具的话题，但交流之中，话题难免发散出去，我也有机会从他们身上学知识、听故事，很多时候聊得投机，从5、6点一直聊到漫咖啡打烊。

虽然这样的夜晚其乐无穷，但过些日子，大部分愉快的记忆就变得氤氲起来，好些人成为了通讯录里的一个陌生ID。因此，我准备启动一个项目：把生命中遇到的有趣的灵魂，用文字的方式记录下来，一来帮我凝固记忆，二来也许也能带给大家一些乐趣。我暂时给它命名为「53号纪录片」，今天是第一篇。
